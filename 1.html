HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render them intomultimedia web pages.

HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML constructs, images and other object, such as interactive forms, may be embedded into the rendered page
